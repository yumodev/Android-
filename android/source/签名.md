## 查看APK签名


```
java -jar signapk.jar platform.x509.pem platform.pk8 apk路径(签名前) apk路径(签名后)
keytool -printcert -file META-INF/CERT.RSA
```

## 查看本地keystore文件

```
keytool -list -v -keystore 目标文件路径
```
