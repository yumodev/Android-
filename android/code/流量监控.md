1、TrafficStats 类

   常用方法：
   getUidRxBytes(uid); //获取一个app的接收流量
   getUidTxBytes(uid); //获取一个app的上传流量

2、通过文件获取流量
   从系统文件中获取一个app的上传流量
       long txBytes = 0;
       try {
          BufferedReader in = new BufferedReader(new FileReader("/proc/uid_stat/" + uid + "/tcp_snd"));
          String line = in.readLine();
          txBytes = Long.parseLong(line);
          in.close();
         }catch(Exception e)
         {
         }
 从文件中获取一个文件的接收流量
