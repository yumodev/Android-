### 10.1 概述

Java语言的编译期是一段不确定的操作过程，因为它可能是一个前端编译期把*.Java文件转换成\*.class的过程；也可以起是后端运行期编译期(JIT编译期)把字节码转变成机器码的过程，还可能是只是用静态提起编译期(AOT编译期)直接把\*.java文件件直接编译成本地及其代码的过程。
