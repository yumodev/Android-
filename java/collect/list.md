### List

1. 有序的Collection集合，能够精确的控制每个元素的插入位置
2. 能够使用索引访问


### 子类

ArrayList,LinkedList, Vector, Stack.

### 面试题目

* Array和ArrayList的区别
* ArrayList和LinkedList的区别，增删操作那个效率比较好。
* ArrayList和Vector的区别。
* ArrayList 是如何扩增内存的。

###  `ArrayList list = new ArrayList(20);` 中的list扩充了几次
